## Vortex Browser v4.8.4

### 📱 Responsive View & Mobile Emulation
Test how websites look on mobile devices directly within Vortex. This update introduces a powerful hardware-level device emulation engine with touch support and auto-scaling.

---

### What's New

**Mobile Device Emulation:**
- **Hardware-Level Integration:** Uses Electron's native device emulation to simulate screen dimensions, device scale factors, and touch event handling.
- **Popular Presets:** Quickly switch between presets for **iPhone 12/13**, **Pixel 5**, and **iPad Air**.
- **User Agent Masking:** Automatically updates the browser's User Agent to match the emulated device, ensuring mobile versions of sites are served correctly.

**Smarter Viewport Handling:**
- **Auto-Scaling Engine:** The emulated viewport automatically scales to fit your current window size, maintaining a realistic device frame regardless of window dimensions.
- **Visual Device Frame:** Added a centered backdrop with a premium dark theme and a realistic device border to provide a focused testing environment.
- **Toggle via Menu:** Easily enable or disable Responsive View from the main navigation menu dropdown.

**🚀 Performance & UI:**
- **Auto-Fit Logic:** Improved the responsiveness of the UI when switching between desktop and mobile views.
- **Touch Event Mapping:** Integrated hardware-level emulation for touch events, making mobile testing more accurate.
