================================================================================
SECTION FILE PATTERN (same as settings/)
================================================================================

Every module follows this pattern:

  ui/*.js      → Pure HTML string functions. No event listeners. No logic.
  scripts/*.js → Event listeners + logic only. No HTML generation.
  index.js     → Assembles ui + binds scripts. Public API exposed.

Example (screenshot):
  screenshotPanel.js  → function buildPanelHTML(dataURL, title) { return `...`; }
  screenshotHandler.js→ function bindEvents(panel) { copy btn, save btn, close btn }
  index.js            → window.Screenshot = { capture, _showLoading }
