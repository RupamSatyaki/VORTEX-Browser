BUGS FIXED (April 2026)
================================================================================

✅ BUG-1: storage:read duplicate handler crash
   Root cause: main.js called Storage.registerStorageHandlers() AND
               IpcHandler.registerHandlers() → storageHandlers.register()
               Both registered storage:read → Electron crash
   Fix: Removed Storage.registerStorageHandlers() from main.js
        Added onWrite(name, cb) callback to storageHandlers.js
        Removed hacky ipcMain.removeHandler('storage:write') override

✅ BUG-2: WhatsAppPanel is not defined
   Root cause: navigation/index.js referenced WhatsAppPanel.toggle()
               but no module existed for it
   Fix: Created features/whatsappPanel.js with open/close/toggle/toggleFullscreen
        Added script tag to index.html before navigation scripts

ℹ️  BUG-3: dragEvent is not defined (console noise only)
   Root cause: Electron internal webview drag implementation bug
               Not from our code — tabDrag.js uses _drag variable correctly
   Status: Harmless — already suppressed in main.js uncaughtException handler
           No functional impact on tab drag feature

ℹ️  BUG-4: All "No handler registered" errors
   Root cause: Cascade from BUG-1 — crash prevented all handlers from loading
   Status: Auto-fixed by BUG-1 fix — all 13 handler files now load correctly
