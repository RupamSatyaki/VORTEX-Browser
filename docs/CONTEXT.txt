KNOWN PATTERNS & CONVENTIONS
================================================================================

1. Panel system: Panel.open('settings'|'downloads'|'bookmarks'|'history')
2. Settings communication: postMessage with __vortexAction flag
3. WhatsApp panel: #wa-panel, persist:whatsapp partition
4. Incognito: separate BrowserWindow, purple theme, ?incognito=1
5. YT Adblock: ipcHandler.js (network) + webview.js (CSS+JS injection)
6. Tab sleep: webview src в†’ 'about:blank', restored on click
7. Version in packaged builds: app.getVersion() NOT require(package.json)
8. CSS toggle class: .toggle-track (NOT .slider)
9. DevHub download: vortexAPI.send('devhub:download', {dataUrl, filename})
10. Modular tools: each tool in own directory, CSS injected via <link> in _injectCSS()
11. Changelog: fetch() based, JSON files per version, major version group dropdowns
12. Cookie Manager: cookieManager.js renders lazily, block list in localStorage
13. Password Manager: dual storage вЂ” vault (encrypted) + imported (plain JSON)
    Autofill: sessionStorage cache for vault, imported always available
    Save prompt: after login/signup via did-navigate-in-page
14. Blocklist: engine.js registers webRequest.onBeforeRequest on startup
    Domain Set in memory, O(1) lookup, per-tab stats via Map
    Settings panel lazy-rendered on nav click
15. Windows Default Browser: installer.nsh NSIS script writes HKLM+HKCU registry
    app.setAsDefaultProtocolClient('http'/'https') in main.js (packaged only)
    open-url IPC в†’ new tab, queued if app not ready (_markAppReady pattern)
16. Custom Alert/Confirm/Prompt: webviewPreload.js overrides window.alert/confirm/prompt
    в†’ ipcRenderer.sendToHost('dialog:show') в†’ ipc-message в†’ VortexDialog.show()
17. new-window в†’ new tab: app.on('web-contents-created') + setWindowOpenHandler
18. Accent color: CSS var(--accent), applied via applyAccent() in app.js
20. Proxy system: src/main/proxy/ вЂ” proxyManager, torProcess, torDownloader, ipcHandlers
    Tor proxy sirf bootstrap 100% ke baad apply hoti hai (onTorReady callback)
    Real IP: session.fromPartition('persist:proxy-ip-check') with proxyRules:'direct://'
    ProxyState: singleton event emitter, clearAll() on panel remount
    Storage: %APPDATA%\vortex\storage\proxy.json
